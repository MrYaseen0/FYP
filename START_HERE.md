## 🎯 The Project: Healtheon

### What Is It?
A **multi-agent AI medical education system** - a university Final Year Project demonstrating how 7 AI agents can collaborate to analyze patient cases using clinical reasoning.

### Key Features
- **7 Interactive Pages** (Dashboard, Form, Detail, Record, Orchestration, Fleet, Analytics)
- **7 Specialized AI Agents** (Intake, GP, Cardiologist, Neurologist, Pulmonologist, Pathologist, Summarizer)
- **Real-Time Monitoring** (Watch agents work live with updating transcript)
- **Professional UI** (Dark glassmorphism with cyan accents)
- **Complete Workflows** (Submission → Analysis → Reporting)

### Technology
- **Frontend**: React 19 + Vite
- **Backend**: FastAPI (Python)
- **Database**: SQLite
- **AI**: Microsoft AutoGen + OpenAI/Ollama
- **Styling**: Tailwind CSS + Custom CSS
