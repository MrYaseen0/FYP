## 🎯 Core Concepts

### The System at a Glance
```
Patient Data → React Form → FastAPI Backend → 7 AI Agents 
→ SQLite Database → Live Transcript → Final Report → Doctor Reviews
```

### The 7 Pages at a Glance
| # | Page | Route | Purpose |
|---|------|-------|---------|
| 1 | Dashboard | / | Case management hub |
| 2 | Patient Form | /submit | Submit new cases |
| 3 | Case Detail | /cases/:id | Real-time processing |
| 4 | Patient Record | /record | Medical record view |
| 5 | System Orchestration | /orchestration | Admin configuration |
| 6 | Agent Fleet | /fleet | Agent management |
| 7 | Case Analytics | /analytics | Case comparison |

### The Agent Pipeline
```
Intake Agent → GP Agent → Specialists (Cardio/Neuro/Pulmo)
→ Pathologist → Summarizer → Final Report
```
